## UI Components Breakdown

### HUD (Heads-Up Display)
Located below game screen:
- **USDC Balance**: Real-time from wallet
- **Pokeball Count**: Current inventory
- **Location**: Current map/area name
- **Quick Actions**: Inventory, Menu buttons

### Status Bar (In-Game)
Top of game screen:
- Player name (wallet ENS or shortened address)
- Time/steps (optional)
- Event notifications

### Dialogue System
- **Box Style**: Classic GameBoy with border
- **Text**: Retro pixel font (Press Start 2P)
- **Speed**: Letter-by-letter animation
- **Controls**: A/Space to continue, B/Esc to skip

### Menu System
- **Navigation**: D-pad or arrow keys
- **Selection**: Highlight + A button
- **Cancel**: B button
- **Animation**: Slide in/out

### Primary Font

  - Tiny: 3px (scaled)

## Animations

### Screen Transitions
- **Fade**: 300ms ease-in-out
- **Slide**: 200ms ease-out
- **Flash**: Instant white flash (for encounters)

### Sprite Animations
- **Walk Cycle**: 4 frames, 150ms per frame
- **Idle**: 2 frames, 500ms per frame
- **Catch**: 6 frames, 200ms per frame

### UI Animations
- **Button Hover**: Scale 1.05, 100ms
- **Selection**: Color change, 150ms
